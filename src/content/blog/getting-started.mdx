Welcome to Hylo! This guide will help you take your first steps with the Hylo programming language.

## What is Hylo?

Hylo is a modern systems programming language that emphasizes safety, performance, and expressiveness. It combines the best features of functional and imperative programming paradigms.

## Installation

To get started with Hylo, you'll need to install the compiler. Visit our [installation guide](/docs) for detailed instructions.

## Your First Hylo Program

Here's a simple "Hello, World!" program in Hylo:

```hylo
fun main() {
    print("Hello, World!")
}
```

## Key Features

### Memory Safety
Hylo provides memory safety without garbage collection through its unique ownership system.

### Performance
Hylo compiles to efficient native code, making it suitable for systems programming.

### Expressiveness
The language includes modern features like pattern matching, generics, and type inference.
